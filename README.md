# ISS Tracker 🛰️

A Python-based web application that tracks the International Space Station (ISS). Input your coordinates and time zone, set a pin on the map, and calculate when the ISS will pass over your location.

## Getting Started

Follow these steps to set up and run the ISS Tracker on your local machine.

### Prerequisites

Make sure you have **Python 3.x** and **git** installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/jacobhart29/ISS-TRACKER](https://github.com/jacobhart29/ISS-TRACKER)
   cd ISS-TRACKER
   ```

2. **Install the required dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

### Running the Application

1. **Start the local server:**
   ```bash
   python main.py
   ```

2. **Open the localhost ip at:**
   ```bash
   http://localhost:8080/
   ```


# How to use

1. Enter your latitude and longitude coordinates.
2. Select your time zone of choice.
3. Drop the pin on the map to confirm your location.
4. Press Calculate to get the time the ISS will fly near/above your selected location.
